Что делает скрипт?

- Позволяет установить порт SSH на выбор
- Создает файл подкачки на 2 gb
- Настроивает DNS over TLS на (Quad9 И Cloudflare)
- Настройка и включение UFW (открывает 2 порта: выбранный SSH и 443/tcp (остальное без изменений))
- Включение автообновления патчей безопасности

- Завершающий этап с очисткой старых зависимостей
