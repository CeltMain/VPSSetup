# Что делает скрипт?

### Базовая, минимальная автоматизация настройки свежей VPS.
</div>

### 🔗 VPS Setup
```bash
bash <(curl -Ls https://raw.githubusercontent.com/CeltMain/VPSSetup/refs/heads/main/vpssetup.sh)
```

- Позволяет установить порт SSH на выбор
- Позволяет создать настраиваемый файл подкачки [Default: 2GB]
- Настройка DNS over TLS [Default: Cloudflare]
- Конфигурация и включение UFW
    + открытие 2 портов: выбранный SSH и 443/tcp (остальное без изменений)
    + отключение создания правил для IPv6
- Включение автообновления патчей безопасности
***
- Завершающий этап с очисткой старых зависимостей

### 🔗 SSH-Tunnel Setup
```bash
bash <(curl -Ls https://raw.githubusercontent.com/CeltMain/VPSSetup/refs/heads/main/tunnelsetup.sh)
```

- Автоматически создает пользователя `sshtunneluser` с заблокированным терминалом (без доступа к консоли сервера)
- Разрешает **одновременное** использование SSH-туннелей (TCP Forwarding) и безопасную передачу файлов по SFTP
- Жестко изолирует (Chroot) сессию пользователя внутри его домашней директории — он физически не может выйти выше своей папки и увидеть системные файлы VPS
- Автоматически создает доступную для записи папку `Download` внутри Chroot-окружения для полноценной работы с файлами по SFTP
- Настраивает безопасный встроенный модуль `internal-sftp`
- Выполняет автоматическую валидацию конфигурации SSH перед перезапуском службы во избежание сбоев
